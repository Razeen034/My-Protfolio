# Rajin Panthee Portfolio

Production portfolio built with Next.js, React, TypeScript, Tailwind CSS, Framer Motion, and Lucide icons.

## Scripts

```bash
npm run dev
npm run lint
npm run build
npm run start
```

## Project Structure

- `src/app` - App Router pages, global styles, metadata, and 404 route
- `src/components` - Portfolio UI components and client-side interactions
- `public` - Static assets served by Next.js

## Production Notes

- Run `npm run lint` and `npm run build` before deployment.
- Keep social/action links pointed at real destinations only.
- Add a real Open Graph image before enabling `openGraph.images` in `src/app/layout.tsx`.
